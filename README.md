Este repositorio contiene la Fase 1 del laboratorio, que consiste en la creación de 4 preguntas de programación relacionadas con análisis de datos y machine learning utilizando pandas y sklearn.

Cada pregunta incluye:

- El enunciado formal del problema.
- Un generador automático de casos de uso que produce entradas aleatorias y el output esperado.
